
Natural Language to IoT Code Generator 🚀

AI-powered framework that converts plain English instructions into executable ESP32 Arduino code with automatic wiring guidance and component recommendations.
Designed to simplify IoT prototyping for beginners, students, and non-technical users using Generative AI + NLP + ESP32.

📌 Project Overview

The Natural Language to IoT Code Generator enables users to create IoT prototypes simply by describing their idea in natural language.

Example:

“Turn on the fan when room temperature exceeds 30°C”

The system automatically:

Detects sensors and actuators
Identifies the IoT domain
Generates ESP32 Arduino code
Provides wiring instructions
Suggests required hardware components

This project aims to make IoT development faster, easier, and accessible to everyone.
